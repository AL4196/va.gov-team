
## SCO Migration Page Prototypes:

### SCO Migration Page initial prototype v0.0 October 3, 2019:
Design is based on DSVA's pattern for a hub page to serve as a launching-off, centralized location for resources for School Certifying Officials and school administrators, and contains accordions to categorize content within the Training & Guides section.
